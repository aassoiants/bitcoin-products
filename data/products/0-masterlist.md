**Scope key** (full criteria at [`docs/product-definition.md`](../../docs/product-definition.md)):
- `bitcoin`: passes the scope gate via code path OR purpose path
- `shitcoin`: built on a non-bitcoin chain (Stacks, RSK, Solana, Ethereum, exSat, BCH, BSV, ICP, Cardano, etc.)
- `unclear`: tagline does not give enough signal; pending review
- `excluded`: evaluated and ruled out (scope failure)
- `withdrawn`: team rescinded their submission and no external evidence remains; kept in masterlist for the internal record but not rendered in public output

**Deep column:**
- `✓`: has a deep-entry file at [`data/products/{slug}.md`](.)
- `-`: breadth-only row
- `excluded`: per the scope gate
