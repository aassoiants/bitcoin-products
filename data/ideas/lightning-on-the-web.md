### Past products (6, 3 alive)

- Bitcoin Connect (2023, lol-2-2023, alive) — One-button Lightning wallet connector for any webapp via NWC; v3.12.2 shipped Feb 2026
- LNbits (2020, fulmo-2020-may + fulmo-2020-jun + fulmo-2020-dec, alive) — Self-hostable Lightning accounts server with LNURL-pay and Lightning Address extensions; v1.0.0 in 2025
- Lightning Browser Extension (2020-2021, fulmo-2020-dec + fulmo-2021-mar, alive) — Browser extension enabling Lightning payments across multiple node implementations; maintained by Alby
- Moneysocket / Costanza (2020-2021, fulmo-2020-dec + fulmo-2021-mar, dormant) — Browser Lightning wallet speaking custom Moneysocket protocol so other apps can drive it without custodying funds
- Lightning Welder (2024, bitcoinpp-ecash-2024, status as captured) — Browser bridge for BOLT-11 wallets to pay BOLT-12 offers
- Fedimint-Web (2024, bitcoinpp-ecash-2024, status as captured) — Browser Lightning/Nostr extension using Fedimint web SDK
