| Field | Value |

| slug | bitcoin-features-in-wallets |
| idea | Add impactful bitcoin features to the wallets people already use. |
| card_body | Reference implementations for bitcoin features ship faster than wallets adopt them. The bottleneck is language bindings: Java for one wallet, Swift for another, Python for a third. Pick a feature, pick a wallet's language, and ship the binding. |
| body | Reference implementations ship faster than wallet adoption. Payjoin is the canonical example: [Payjoin Dev Kit](https://payjoindevkit.org/) exists, and [Bull Bitcoin](https://www.bullbitcoin.com/) and [Cake Wallet](https://cakewallet.com/) have shipped v2 integrations, but that's two wallets. The work is mostly language bindings: Java for Sparrow, Swift for FullyNoded, C++ for Bitcoin Core and Nunchuk, Python for Electrum, Dart for Aqua. Same shape applies to other features waiting for bindings. |
